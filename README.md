# @younho9/tsconfig

> Shared TypeScript config that personally prefer

## Install

```sh
npm install --save-dev @younho9/tsconfig
```

## Usage

`tsconfig.json`

```json
{
	"extends": "@younho9/tsconfig/tsconfig.json",
	"compilerOptions": {
		// override options
	}
}
```
